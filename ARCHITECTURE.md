 *   └────────┬────────┘
 *            │
 *            ├──────────────────┬──────────────────┬─────────────────┐
 *            │                  │                  │                 │
 *            ▼                  ▼                  ▼                 ▼
 *     ┌──────────┐       ┌──────────┐       ┌──────────┐     ┌──────────┐
 *     │ style.css│       │  app.js  │       │ diaries  │     │ wishes   │
 *     │ (全局样式)│       │ (主逻辑)  │       │ .json    │     │ .json    │
 *     └──────────┘       └────┬─────┘       └──────────┘     └──────────┘

 *                             │                                   │

 *                     ┌──────────────┐                   ┌──────────────┐

 *                     └──────────────┘                   └──────────────┘
 * 
 * 【页面结构】
 * 
 * 1. index.html (首页)
 *    - 密码锁
 *    - 恋爱计时器
 *    - 纪念日倒计时
 *    - 精选情书
 *    - 照片预览
 *    - 愿望清单进度
 * 
 * 2. writings.html (情书博物馆)
 *    - 年度筛选
 *    - 关键词搜索
 *    - 情书卡片列表
 *    - 情书详情弹窗
 *    - 年度电子书生成
 * 
 * 3. photos.html (照片墙)
 *    - 瀑布流布局
 *    - 灯箱浏览
 *    - 照片上传 (需认证)
 *    - 分类筛选
 * 
 * 4. travels.html (足迹地图)
 *    - Leaflet 地图
 *    - 足迹标记点
 *    - 时间轴联动
 *    - 足迹添加 (需认证)
 * 
 * 5. wishes.html (愿望清单)
 *    - 愿望列表
 *    - 完成状态切换
 *    - 进度统计
 * 
 * 6. time-capsule.html (时间胶囊)
 *    - 未来留言
 *    - 时间解锁
 * 
 * 【JS 模块说明】
 * 
 * ┌──────────────────┬──────────────────────────────────────────────────┐
 * │ 模块文件           │ 功能说明                                         │
 * ├──────────────────┼──────────────────────────────────────────────────┤
 * │ app.js           │ 首页主逻辑：计时器、密码锁、滴答同步、localStorage │
 * │ simple-auth.js   │ GitHub PAT 认证：密码+PAT 双重验证                │
 * │ simple-edit.js   │ 编辑模式控制：密码 2025、显示/隐藏编辑按钮        │
 * │ editor.js        │ 内容编辑：照片/情书/足迹/愿望的增删改查           │
 * │ auth.js          │ GitHub OAuth 登录（可选）                        │
 * │ edit-ui.js       │ 编辑界面 UI 组件                                 │
 * └──────────────────┴──────────────────────────────────────────────────┘
 * 
 * 【数据存储】
 * 
 * ┌──────────────────┬──────────────────────┬────────────────────────────┐
 * │ 数据文件          │ 内容                  │ 存储位置                   │
 * ├──────────────────┼──────────────────────┼────────────────────────────┤
 * │ diaries.json      │ 32篇情书              │ GitHub仓库 data/          │
 * │ photos.json      │ 照片元数据            │ GitHub仓库 data/          │
 * │ travels.json      │ 足迹标记点            │ GitHub仓库 data/          │
 * │ wishes.json      │ 愿望清单              │ GitHub仓库 data/          │
 * │ localStorage      │ 用户设置、密码、主题   │ 浏览器本地                 │
 * └──────────────────┴──────────────────────┴────────────────────────────┘
 * 
 * 【GitHub API 操作流程】
 * 
 * 1. 用户输入密码 2025 进入编辑模式
 * 2. 用户输入 GitHub PAT
 * 3. PAT 存储到 localStorage
 * 4. 编辑内容时，通过 editor.js 调用 GitHub API
 * 5. API 操作：GET(读取SHA) → PUT(提交内容)
 * 
 * 【CSS 变量】
 * 
 * ┌─────────────────┬────────────┬─────────────────────────────────────┐
 * │ 变量名            │ 默认值      │ 用途                                │

 * │ --accent        │ #C4A8A2    │ 主色调（莫兰迪粉）                   │
 * │ --accent-light  │ #E2C5C5    │ 浅色调                             │
 * │ --bg-primary    │ #FDFBF7    │ 主背景色                           │
 * │ --text-dark     │ #5a4a4a    │ 主文字色                           │
 * │ --radius-sm     │ 8px        │ 小圆角                             │
 * │ --radius-md     │ 16px       │ 中圆角                             │
 * │ --radius-lg     │ 24px       │ 大圆角                             │
 * └─────────────────┴────────────┴─────────────────────────────────────┘

 * 【图标规范】
